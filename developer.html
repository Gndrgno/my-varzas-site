Важно

  1. Основания для обращения:
    1. Сообщение о технических ошибках или багах
    2. Предложения по улучшению функционала
    3. Вопросы по работе существующих функций
    4. Сообщения о проблемах безопасности
  2. Запрещенные обращения:
    1. Личные вопросы, не связанные с проектом
    2. Жалобы на действия модерации/администрации
    3. Коммерческие предложения и реклама
    4. Оскорбительные или троллинговые сообщения
  3. Права разработчика:
    1. Отказать в ответе на нарушающие правила обращения
    2. Игнорировать неинформативные сообщения
    3. Блокировать пользователей за нарушения
    4. Перенаправлять запросы не по своей компетенции
  4. Условия обработки обращений:
    1. Срок ответа - 1-5 рабочих дней
    2. Отсутствие гарантий реализации предложений
    3. Требование конструктивной формы обращений

Отправляя сообщение, вы подтверждаете согласие с этими правилами.
